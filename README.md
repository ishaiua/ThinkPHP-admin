## 简介

基于ThinkPHP3.2的多模块后台管理系统

使用

1.git clone https://github.com/ishaiua/ThinkPHP-admin.git

2.在该项目目录运行： php -S localhost:8000

3.在浏览器中输入：localhost:8000
